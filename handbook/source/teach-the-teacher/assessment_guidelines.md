# Assessment Guidelines

Evaluation in a Teach-the-Teacher program focuses on **understanding, demonstration, and reflection**, not only tool usage.

## Evaluation Criteria

| Category | Description | Example Evaluation Criteria |
|:--|:--|:--|
| **Conceptual Understanding** | Ability to explain the role and dependencies of each stage. | “Describe the difference between synthesis and placement.” |
| **Technical Execution** | Correctly running synthesis, routing, and verification steps. | Successfully obtaining a DRC/LVS-clean GDS file. |
| **Troubleshooting Skills** | Identifying and fixing common issues. | Resolving library path or netlist mapping errors. |
| **Collaboration and Contribution** | Participation and documentation sharing. | Submitting improvements or additional exercises. |
